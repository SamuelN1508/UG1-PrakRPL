# Trial UnGuided 
Buatlah sebuah program Java yang digunakan untuk menerima input berupa nama (String), umur (int), dan nilai ujian (int) dari pengguna. Program kemudian menampilkan hasil evaluasi yang berisi nama, umur, nilai ujian, grade, serta status kelulusan berdasarkan nilai yang dimasukkan.

## Peraturan:
Jika nilai tidak berada dalam rentang 1–100, maka program menampilkan pesan:
Nilai tidak valid!
Nilai harus berada di antara 0 - 100

## Ketentuan grade nilai:
- A : 85 – 100
- B : 75 – 84
- C : 65 – 74
- D : 50 – 64
- E : < 50

## Ketentuan status kelulusan:
Nilai ≥ 75 → LULUS

Nilai < 75 → TIDAK LULUS

### Contoh Output
```python
Masukkan nama           : Andi
Masukkan umur           : 19
Masukkan nilai ujian    : 80

===== HASIL EVALUASI =====
Nama   : Andi
Umur   : 19
Nilai  : 80
Grade  : B
Status : LULUS
```

### Jika Nilai tidak valid:

```python
Nilai tidak valid!
Nilai harus berada di antara 0 - 100
```
